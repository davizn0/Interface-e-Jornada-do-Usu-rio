# Aplicativo - Vida Social

## Descrição:
O objetivo principal da tarefa seria lançar uma função de criar evento sociais dentro de uma plataforma
de redes sociais. O formato da designer do aplicativo WhatsApp foi usado para desenvolver o projeto.
A tarefa principal é o usuário criar um evento, convidar pessoas e acompanhar confirmações de presença.
