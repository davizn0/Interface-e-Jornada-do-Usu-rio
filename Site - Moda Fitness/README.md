# Site - Moda Fitness

## Descrição:
O protótipo vem com o objetivo principal de ser um portal de atendimento de serviços. Sendo assim, foi criado um site
onde oferece roupas, do estilo fitness, para os clientes poderem visualizar produtos digitalmente e realizar compras.
Como também, é possível visualizar o status de pedido/entrega e também podem realizar e verificar avaliações de outros clientes.
